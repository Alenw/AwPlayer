# AwPlayer
A video player can play with online URL or offline document path .

###### 一个对AVFoundation 进行封装的播放器，支持的视频编码格式有：H.264、MPEG-4，扩展名（压缩格式）：.mp4、.mov、.m4v、.m2v、.3gp、.3g2等 
<br/>
###### 如果想支持更多的格式，可以使用我另一份集成FFMPEG代码，支持各种流媒体视频格式；
